Like UI

audio标签属性

src：音频的URL

preload：预加载

autoplay：自动播放

loop：循环播放

controls：浏览器自带的控制条

video标签属性

src：视频的URL

poster：视频封面，没有播放时显示的图片

preload：预加载

autoplay：自动播放

loop：循环播放

controls：浏览器自带的控制条

width：视频宽度

height：视频高度

创建对象

audio使用new Audio(src)创建对象；

audio、video都可以使用document.getElementById("media")获取对象；或者$("#Media")[0]

Media方法

addTextTrack(kind,label,language)向Media添加新的文本轨道

canPlayType(type)是否能播放某种格式的资源

load()重新加载src指定的资源

play()开始播放Media

pause()暂停播放Media

Media属性

autoplay设置或返回是否在加载完成后随即播放音频/视频 boolean

buffered返回音视频的已缓冲部分 TimeRanges

controller返回音视频的当前媒体控制器 MediaController

controls设置或返回浏览器应当显示标准的音视频控件 boolean

currentSrc熟悉返回当前音频/视频的 URL string

currentTime设置或返回音频/视频中的当前播放位置（以秒计） int

defaultMuted设置或返回音频/视频默认是否静音 boolean

defaultPlaybackRate设置或返回音频/视频的默认播放速度 int

duration返回当前音频/视频的长度（以秒计） int

ended返回音频/视频的播放是否已结束 boolean

error返回音频/视频的错误状态 MediaError

loop设置或返回音频/视频是否应在结束时重新播放 boolean

mediaGroup设置或返回音频/视频所属的组合（用于连接多个音频/视频元素） string

muted设置或返回音频/视频是否静音 boolean

networkState返回音频/视频的当前网络状态 int

paused设置或返回音频/视频是否暂停 boolean

playbackRate设置或返回音频/视频播放的速度 playbackspeed

played返回表示音频/视频已播放部分的 TimeRanges 对象 TimeRanges

preload设置或返回音频/视频是否应该在页面加载后进行加载 string

readyState返回音频/视频当前的就绪状态 int

seekable返回表示音频/视频可寻址部分的 TimeRanges 对象 TimeRanges

seeking返回用户目前是否在音频/视频中寻址 boolean

src设置或返回音频/视频元素的当前来源 string

startDate返回表示当前时间偏移的 Date 对象 date

textTracks返回表示可用文本轨道的 TextTrackList 对象 TextTrackList

videoTracks返回表示可用视频轨道的 VideoTrackList 对象 VideoTrackList

volume设置或返回音频/视频的音量 int 当前音量： -
